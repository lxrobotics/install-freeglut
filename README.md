:floppy_disk: `install-freeglut`
===============================

This repository contains a GitHub Action to install [FreeGLUT](https://github.com/freeglut/freeglut).

#### How-to-use
```bash
steps:   
  - name: Install FreeGLUT
    uses: lxrobotics/install-freeglut@main
```
